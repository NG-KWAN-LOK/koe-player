# 日文作業出題指引

## 背景

這份指引是為了讓 AI 模仿原日文老師的出題風格，為學生出**日文聽力作業**。
學生程度：中高級（能應付日常口語、理解複雜句型、表達細膩感情和意見）

**最重要原則：所有題目必須生活化，用真實日常會話中真正用到的表達，讓學生可以直接應用在生活中。避免過於教科書或書面的句子。**

---

## 出題方式

AI 生成題目文字後，將文字轉成錄音傳給學生。學生**只能聽錄音**，不能看到題目文字，需要靠聽力完成作業。

## 作業流程

### 出題

1. AI 生成題目文字（注文、会話、カタカナ 三個部分）
2. **題目文字不能讓學生看到**——題目寫成 `.md` 檔保存，TTS 用 JSON 另存成 `.json` 檔，只將 `.json` 檔傳給學生生成語音
3. 學生只聽錄音，手寫日文及中文翻譯（中文翻譯可以用廣東話）

**TTS JSON 格式：**

```json
{
  "lines": [
    {
      "id": "{section}-{序號}",
      "lang": "ja-JP | yue-HK",
      "gender": "F | M",
      "text": "該句文字"
    }
  ]
}
```

- `id` prefix：`order-` = 注文、`conv-` = 会話、`kata-` = カタカナ、`order-ans-` / `conv-ans-` = 中文句子的日文建議答案
- 廣東話句子用 `yue-HK`，日文用 `ja-JP`
- `gender`：`"F"` = 女聲、`"M"` = 男聲（注文一律用 `"M"`、老師 B 的台詞用 `"M"`）
- 每句獨立一個 object，按出題順序排列
- **会話中被替換成中文的句子，其日文原文作為建議答案放在 JSON 最末尾**，id 用 `conv-ans-1`、`conv-ans-2`…（從 1 遞增）

### 交功課

3. 學生將手寫作業**拍照**傳給 AI
4. 學生將日文部分**錄音**傳給 AI

### 批改

5. AI 先讀取該次作業的題目 `.md` 檔，對照正確答案
6. AI 看照片，批改日文書寫、漢字上的平仮名標註是否正確、以及中文翻譯
7. AI 聽錄音，批改日文發音和口音
8. 提供正確答案時，所有漢字必須標註讀音（例：枝豆（えだまめ）、花火大会（はなびたいかい））

## 翻譯批改原則

所有部分的中文翻譯只需意譯，能傳達原本意思即可，不執著字眼，但文法必須正確。

## 作業內容（三個部分）

### 1. 注文

AI 出一道日文點餐句，學生要：

- 手寫日文
- 漢字標平仮名
- 中文翻譯（廣東話也可以）

### 2. 会話

AI 出一段 A、B 兩人的日文對話，學生要：

- 手寫日文
- 漢字標平仮名
- 中文翻譯（廣東話也可以）

另外，AI 完成完整對話後，再從中抽出 1〜2 句較簡單的句子，改用中文出題，學生要自行翻譯成日文。
**注意：先生成完整、符合邏輯的對話，之後才決定抽哪句。不要因為要抽句而故意把對話出得簡單。**
**出題時，中文句子必須直接替換到対話裡對應的位置，不可以另起一段。講稿的対話要日文和中文混在一起，例如：**

A：ねね、来週の土曜日の花火大会、もう準備した？
B：まだだよ。何か持って行った方がいい物ある？
A：野餐墊同飲料一定要帶㗎，因為上年忘記帶搞到好狼狽。
B：確かに。去年は立ちっぱなしだったよね。
A：そうそう。あと、浴衣着る？
B：嗯，想著嘅，但自己一個人著唔到呀。

### 3. カタカナ

AI 出三個外來語詞彙，學生要：

- 手寫カタカナ
- 中文或英文翻譯

**出題時只需列出三個詞語，不用附中文或英文翻譯。**

## 注文出題規則

**句型固定：**
すみません、〜を一つと、〜を一つ（と）、〜を一つください。

**每次必須包含：**

- 1〜2 款飲料 ＋ 2〜3 款食物
- 總品數 3〜5 品

**場景設定（每次隨機抽選一種，不要重複）：**

- 居酒屋
- 焼き鳥屋・串焼き屋
- 回転寿司・寿司屋
- おでん屋・屋台
- ラーメン屋
- 定食屋・食堂
- 焼肉屋
- 天ぷら屋
- うどん屋・そば屋
- カフェ・喫茶店
- ファミレス（ファミリーレストラン）
- お好み焼き屋・たこ焼き屋

**飲料選擇（配合場景，以下為例，不限於此）：**

- 居酒屋・焼肉系：ハイボール、梅酒のロック、生ビール、レモンサワー、芋焼酎の湯割り／水割り、熱燗、泡盛、日本酒
- カフェ・喫茶店系：アイスコーヒー、カフェラテ、抹茶ラテ、紅茶、ほうじ茶ラテ
- 食堂・定食系：お茶、味噌汁、瓶ビール
- ファミレス系：ドリンクバー、メロンソーダ、ジンジャーエール、オレンジジュース

**食物分類（配合場景，以下為例，不限於此）：**

- 居酒屋系：枝豆、唐揚げ、焼き餃子、ポテサラ、冷奴、豚キムチ、たこわさ
- 焼き鳥・串焼き系：せせり、鶏のタレ、皮の塩、やげん軟骨、豚バラ、牛タン、手羽先
- 寿司・海鮮系：ホタテ、いくら、中トロ、鰻、サンマ、イワシ、タコ、いか、ネギトロ
- おでん系：大根、はんぺん、がんも、こんにゃく、牛筋、ロールキャベツ
- 麺・ご飯系：ざるうどん、とんこつラーメン、焼きそば、チャーハン、おにぎり
- 焼肉系：カルビ、ハラミ、タン塩、ホルモン、ビビンバ、クッパ、ナムル
- 天ぷら系：海老天、かき揚げ、野菜天盛り、天丼、天ぷらうどん
- カフェ系：チーズケーキ、パンケーキ、フレンチトースト、クロワッサン、サンドイッチ
- ファミレス系：ハンバーグ、オムライス、エビフライ、ドリア、カルボナーラ
- お好み焼き系：豚玉、ミックス焼き、もんじゃ焼き、たこ焼き、焼きそば

**注意：注文的食物和飲料類型要配合場景設定和当日的会話主題**
例：会話聊焼き鳥 → 場景選焼き鳥屋；会話聊沖縄 → 居酒屋＋泡盛＋沖縄料理；会話聊カフェ → 場景選カフェ

## 会話出題規則

**場景：** 朋友之間的日常閒聊（居酒屋、街上、日常）
**人物：** A = 任何人（朋友、學生、家人、店員等），B = 老師（男）
**來回：** 3〜6 句（對話要自然流動）

**對話結構：**

1. A 拋出話題或觀察
2. B 回應（可主動延伸話題）
3. A 追問細節或表達意見
4. B 回答並加入自己的感想或反問
5. A 再回應
6. （繼續自然流動，不一定有固定終點）

**話題類型（每次隨機抽選一種，自由選擇，以下為例，不限於此）：**

- 天氣・季節：雨、台風、梅雨、花粉症、猛暑日、桜が咲いた、紅葉見に行こう
- 週末・休日計劃：バーベキュー、週末何する、連休の予定、日帰り温泉、家でゴロゴロ
- 老師的私事：住所、通勤時間、家族、做飯、飲み会、ペットの話、趣味
- 旅行計劃：計劃 → 準備 → 出發 → 回來感想，目的地輪換（オーストラリア、北海道、台湾、中国、沖縄、京都、韓国、タイ）
- 旅行トラブル：飛行機が遅れた、ホテルの予約が消えた、道に迷った、パスポート忘れた
- 新聞・SNS：ニュース見た？、インスタ見たよ、Xでトレンド入り
- 身邊的人：生徒・友達的轉職、旅行、誕生日、結婚、引っ越し、出産
- 實用對話：餐廳預約、交通（電車遲延、乗り換え）、送金、宅配便
- 人生觀・意見：お金があったら、もし〜だったら、一番大切なこと、老後の計画
- 比較・選択：A と B どっちがいい？、〜より〜の方が、都会と田舎どっちに住みたい
- 食物・グルメ：最近行ったお店、おすすめのラーメン屋、自炊 vs 外食、コンビニの新商品
- 健康・運動：ジム通い、ダイエット、筋トレ、ランニング、肩こり・腰痛、健康診断の結果
- 買い物・お金：セール、ネットショッピング、衝動買い、節約術、ポイントカード、サブスク
- 仕事・職場：残業、上司の話、転職、リモートワーク、会議が多すぎ、有給取りたい
- 学校・勉強：テスト、宿題、受験、留学、資格試験、勉強法
- 趣味・娯楽：映画、ドラマ、アニメ、漫画、ゲーム、読書、カメラ、釣り、キャンプ
- 音楽・ライブ：好きなバンド、ライブに行った、フェス、カラオケ、新曲
- スポーツ：野球、サッカー、推しのチーム、オリンピック、試合の結果
- ペット・動物：犬派 vs 猫派、ペットの世話、動物園、水族館、かわいい動物の動画
- 住まい・引っ越し：部屋探し、家賃、一人暮らし、インテリア、隣の部屋がうるさい
- テクノロジー：新しいスマホ、アプリのおすすめ、AI、パソコンが壊れた、Wi-Fi が遅い
- 恋愛・人間関係：好きなタイプ、デートスポット、友達と喧嘩した、久しぶりに再会
- 文化・イベント：お祭り、花火大会、クリスマス、お正月、ハロウィン、バレンタイン
- 失敗談・面白い話：やらかした話、恥ずかしかった話、変な夢を見た、電車で寝過ごした

**中高級固定句型（要定期重複練習）：**

確認／追問：

- 〜って本当に？（真的嗎）
- 〜って決まった？（決定了嗎）
- もう〜した？／まだ？（做了嗎／還沒）
- それで、どうなったの？（然後呢）
- え、マジで？（認真的？）

引出話題：

- そういえば、〜（對了，說起來）
- ねね、〜についてちょっと相談してもいい？（可以請教你嗎）
- 〜って聞いたんだけど、本当？（我聽說，是真的嗎）

表達感想與意見：

- 正直〜だと思う（老實說我覺得）
- 〜って感じがする（感覺好像）
- 〜って意外だよね（意外吧）
- めっちゃわかる（超有同感）
- 確かに〜だけど、〜（確實，但是）

假設句：

- もし〜だったら、どうする？（如果是的話你怎辦）
- 〜ばよかった（早知道就好了）
- 〜かもしれない（說不定）

傳聞：

- 〜らしいよ（聽說）
- 〜んだって（人家說）

理由與解釋：

- 〜から、〜んだよ（因為，所以）
- 〜てしまった（不小心做了／後悔）
- 〜わけじゃないけど（不是說，但是）

變化：

- 最近〜ようになった（最近開始）
- 前より〜になった（比以前更）

請求：

- 〜てくれない？（可以幫我嗎，口語）
- 〜てもらえる？（可以幫我嗎，客氣）

## カタカナ出題規則

每次出 3 個，要附中文或英文翻譯。

**類型輪流出現（以下為例，不限於此）：**

- 外來語食物／飲料：ポテトサラダ、アスパラガス、カリフラワー…
- 服飾：カーディガン、スカーフ、スニーカー…
- 國家・地名：カリフォルニア、フィンランド、デンマーク…
- 品牌・人名：カルティエ、エルメス、ロレックス…
- 科技・日常用品：パソコン、プリンター、ソフトウェア…
- 運動：テニス、バレーボール、バドミントン…
- 娛樂・文化：セーラームーン、デジモン、ウルトラマン…

**注意：三個詞彙最好同一個主題或相關**

## 中高級出題注意事項

1. 注文的食物和飲料要是真實居酒屋、餐廳常見的，學生去日本真的會點到的
2. 会話來回要更自然流動，B 不只是回答，也可以主動說話或反問
3. 句型不要太直接，多用口語縮約形（〜てる、〜じゃん、〜だよね）
4. 定期在会話中加入中高級句型，但必須是日常口語自然的用法，不要為了用句型而用
5. 話題要貼近真實生活，學生學完之後能直接在日常對話中用到
6. 話題可以在同一段会話中自然轉換，不限只有一個主題
7. 參考「日文作業範例.md」確保整體風格一致
